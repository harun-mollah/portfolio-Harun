## 🎬 Animation Examples

```
HERO SECTION:
  • Fade in heading: 0.8s
  • Slide in buttons: 0.8s delay
  • Float background: continuous 3s
  • Bounce scroll indicator: continuous

ABOUT SECTION:
  • Slide in from left: 0.7s
  • Stats cards appear: staggered 0.1s
  • Feature cards appear: staggered 0.1s
  • Hover: lift up 10px

PROJECTS SECTION:
  • Card appear: staggered 0.1s
  • Hover: lift up 8px
  • Icons rotate: 10° on hover

SKILLS SECTION:
  • Progress bars fill: 1s each
  • Cards appear: staggered 0.1s
  • Hover: scale 1.05

EXPERIENCE SECTION:
  • Timeline cards appear: staggered 0.2s
  • Cards move right on hover
  • Icons glow on hover
```

## 🎨 Typography

```
HEADINGS:
- H1 (Hero): 3.5rem (mobile) / 5rem (desktop)
- H2 (Section): 2rem (mobile) / 3rem (desktop)
- H3 (Cards): 1.25rem
- H4 (Subsections): 1rem

BODY:
- Large: 1.125rem (18px)
- Regular: 1rem (16px)
- Small: 0.875rem (14px)
- Tiny: 0.75rem (12px)

FONT FAMILY:
- System fonts for best performance
- Fallback to sans-serif

LINE HEIGHT:
- Headings: 1.2
- Body: 1.6
- List items: 1.8
```

- Float: ±20px vertical, 3s duration
